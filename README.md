# Tomer DayBoard

אפליקציית לוח יום בעברית עם משימות, תזונה, מים, משקל והיסטוריה.

## מה חדש בגרסה הזו
- תיקון הבאג בשדה "איפה המשימה עומדת" במשימות.
- הוספת עדיפות למשימות: רגילה / דחופה / נמוכה.
- סיכום משימות מהיר: פתוחות, היום, דחופות, באיחור.
- סינון משימות לפי פתוחות / היום / דחופות / בוצעו / הכל.

## הרצה מקומית
```bash
npm install
npm run dev
```

## Deploy ב-Vercel
Framework: Vite
Build Command: npm run build
Output Directory: dist
