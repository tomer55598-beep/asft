# Tomer Dayboard

גרסה מעודכנת:
- בתזונה אפשר להוסיף מהר רק שם וכמות, בלי להזין קלוריות/חלבון/שומן.
- אפשר עדיין להוסיף עם פרטים אם יש נתונים מהאריזה או ממוצר שמור.
- מוצר שנשמר בספר מוצרים עם פרטים יזוהה בשדה התזונה וימלא את הפרטים אוטומטית.
- מוצר שמור בלי פרטים יופיע בהשלמה וניתן להוסיף אותו מהר בלי מספרים.
- תוקן להתאמה ל-Vercel בלי package-lock שנוצר מסביבה פנימית.

הרצה מקומית:
```bash
npm install
npm run dev
```

Build:
```bash
npm run build
```
